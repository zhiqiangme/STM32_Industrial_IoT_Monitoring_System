# STM32 Mill 控制系统

基于 STM32F1xx 的工业控制系统，支持多传感器采集、继电器控制、远程维护和本地升级。

## 硬件配置

- **MCU**: STM32F1xx (Cortex-M3, 72MHz)
- **通信**:
  - USART2: Modbus RTU 主站 (连接传感器)
  - USART3: Modbus RTU 从站 (供 G780S 远程模块读取)

## 功能特性

### 传感器采集
| 设备 | 从站地址 | 说明 |
|------|---------|------|
| PT100 温度传感器 | 0x01 | 4 通道温度采集 |
| 称重模块 | 0x03 | 4 通道重量采集 (CH3 实际使用) |
| 16路继电器模块 | 0x02 | 16 路输出 + 16 路输入 |
| 流量计 | 脉冲输入 | 瞬时流量和累计流量 |

### 控制功能
- 继电器输出控制 (16 路)
- 继电器输入监测 (16 路, 带去抖)
- 按键本地控制
- 远程云端控制

### 远程维护
- 通过 G780S 模块实现远程参数配置
- 远程配置参数存储到 Flash
- 支持远程升级请求

### 固件升级
- Bootloader 模式支持本地升级
- 本地升级传输协议使用 YMODEM
- 升级状态镜像存储

## 配套工具

### Modbus_CRC
快捷生成 Modbus RTU 原始帧的小工具，用于计算 Modbus CRC16 校验码并生成完整的指令帧。

### Windows 上位机
基于 .NET WPF 的上位机程序，用于连接串口显示传感器数据。
- 使用 NModbus 库进行 Modbus 通信
- 支持串口连接、温度/重量/继电器状态显示

**注意**: 上位机程序已长时间未更新，部分功能可能与当前 STM32 固件不完全同步，建议配合串口调试工具使用。

## 寄存器定义

详细寄存器定义请参考 `远程维护指令手册.md`。

### 现场数据寄存器 (0x0000 - 0x0015)
- 上报序号、温度、重量、流量、继电器状态

### 远程配置寄存器 (0x0020 - 0x0028)
- 传感器采集周期、流量采样周期、DI去抖时间
- 温度变化阈值、流量换算参数、控制模式

### 维护控制寄存器 (0x0030 - 0x0037)
- 解锁口令、维护命令、状态、错误码

### 诊断寄存器 (0x0038 - 0x0059)
- 固件版本、运行时长、上电次数、最近一次保存快照、升级快照、通信异常分类等
