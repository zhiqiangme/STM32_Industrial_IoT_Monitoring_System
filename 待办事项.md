# STM32 Mill 待办事项

本文档把当前后续工作分成四类：

- 已完成：已经落地到当前代码
- 建议做：当前阶段值得推进，投入产出比高
- 可后置：有价值，但不必马上做
- 当前不建议优先做：不是不能做，而是现在做收益不高或容易把重点带偏

## 一、已完成

### 1. 远程参数维护补强

已完成内容：

- 为远程参数增加更严格的上下限检查
- 对关键 16 位参数在单寄存器写入时直接拦截非法值
- 对关键参数禁止写入 0 或超范围值
- 在提交保存前做整体一致性校验
- 细化维护错误码分类

当前已落地的组合校验：

- `flow_sample_period_ms <= sensor_period_ms`
- `di_debounce_ms < sensor_period_ms`

说明：

- 16 位参数会在写入当下校验
- 32 位参数因为拆高低寄存器写入，范围检查会在提交保存时统一校验

- 第二阶段 Bootloader 与本地升级框架已经完成
